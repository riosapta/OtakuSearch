OTAKU SEARCH

OtakuSearch is a light novel / manga / anime database application like myanimelist.net or novelupdates.com but in this directory, the user can search if the series has LN and manga, or manga and anime, or all three. So it's not exclusive to anime only, or just novels like the ones that already exist.
